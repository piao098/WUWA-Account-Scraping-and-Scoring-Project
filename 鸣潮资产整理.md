# 鸣潮账号资产整理

本文档用于后续完善 `configs/scoring_rules.json`。它先把鸣潮角色按大版本、定位和专武整理出来，再列出抽卡资源、命座/武器精炼价值讨论、当前螃蟹网站可爬取字段，以及一个具体账号样例。

说明：

- `1.x / 2.x / 3.x` 按角色首次上线大版本粗分。
- 定位分为 `主C`、`副C`、`辅助`。部分角色可双定位，表中按账号估值时更常用的价值定位归类。
- 3.x 角色和部分新武器处于持续更新期，表中标注 `待校验` 的项后续需要用官方公告、WIKI 或游戏内资料复核后再写入正式评分配置。

## 抽卡相关资源

当前螃蟹详情接口中可稳定解析的抽卡/兑换资源：

| 资源 | 字段名 | 估值思路 |
| --- | --- | --- |
| 星声 | `resources.星声` | 主要抽卡货币，通常按 `160 星声 = 1 抽` 折算。 |
| 月相 | `resources.月相` | 付费货币，可按兑换比例折算成抽数或现金价值。 |
| 浮金波纹 | `resources.浮金波纹` | 角色池抽卡券，直接按抽数计。 |
| 铸潮波纹 | `resources.铸潮波纹` | 武器池抽卡券，直接按抽数计。 |
| 余波珊瑚 | `resources.余波珊瑚` | 商店兑换资源，可折算为额外价值，但不应完全等同抽数。 |
| 黄数 | `yellow_count` | 账号累计五星资产粗指标，不等于抽数，但对强度和投入量有参考价值。 |

后续可以新增两个派生字段：

- `estimated_character_pulls = 星声 / 160 + 浮金波纹`
- `estimated_weapon_pulls = 星声 / 160 + 铸潮波纹`

如果要更严谨，可以把星声拆成“可用于任意池”的通用抽数，而不是同时计入角色池和武器池。

## 命座与武器精炼价值讨论

### 角色命座

| 命座段 | 建议价值处理 |
| --- | --- |
| 0链 | 角色拥有价值，限定主C/辅助按角色本体计分。 |
| 1链 | 小幅加分；若是关键机制命座，可单独提高。 |
| 2链 | 通常是重要节点，主C和顶级辅助应明显加分。 |
| 3链 | 中高投入账号信号，增值明显但开始看边际收益。 |
| 4-5链 | 高投入账号，市场溢价存在但买家范围收窄。 |
| 6链/满命 | 顶级溢价，尤其是新版本主C、强辅助、热门角色；但对常驻角色应打折。 |

建议评分模型：

- 限定主C：0链基础高，2链/6链额外高。
- 辅助：0链价值高，关键命座加成可超过普通主C命座。
- 常驻角色：高命加分，但倍率低于限定角色。
- 老版本主C：随版本衰减，命座价值应低于新版本同档角色。

### 武器精炼

| 精炼段 | 建议价值处理 |
| --- | --- |
| 精1 | 拥有专武的核心价值，角色+专武配套应加成。 |
| 精2-3 | 中等加分，适合高预算账号。 |
| 精4-5 | 高氪信号，但边际收益和市场接受度都要谨慎。 |

建议评分模型：

- 先看“角色是否有对应专武”，再看精炼。
- 精炼高但没有对应角色，按泛用武器价值折价。
- 常驻五星武器重复较多时，不应和限定专武等价。
